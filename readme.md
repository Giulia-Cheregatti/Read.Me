 ## 📌 O que é o Read.Me?
 
**Read.Me** é uma plataforma web para prática de **Leitura em Inglês**, voltada a falantes de português brasileiro. O objetivo é oferecer uma experiência de aprendizado estruturada, agradável e completamente acessível.

O projeto adota o sistema **CEFR** (*Common European Framework of Reference for Languages*) como base pedagógica, organizando o conteúdo em seis níveis progressivos – do **A1** (iniciante absoluto) ao **C2** (maestria quase nativa).

## ✨ Funcionalidades Implementadas na 1ª Etapa

- **Interface bilíngue** — PT-BR ↔ EN com troca dinâmica, sem recarregar a página
- **Dark / Light mode** — alternância com preferência salva no `localStorage`
- **6 níveis CEFR** — A1, A2, B1, B2, C1 e C2 com cards individuais e barras de progresso
- **12 textos autênticos** — 2 por nível, entre 250 e 650 palavras cada
- **96 questões de quiz** — múltipla escolha, verdadeiro/falso e fill-in-the-blank
- **Feedback imediato** — explicação pedagógica para cada resposta
- **Dicionário por clique** — palavras-chave clicáveis com tradução em tooltip
- **Chips de vocabulário** — 5 a 10 palavras-chave por texto, exibidas antes da leitura
- **Painel de progresso** — streak diário, média de pontuação e histórico recente
- **Notificações toast** — feedback visual leve para ações do usuário
- **Roteamento SPA** — navegação via `#hash` sem recarregar a página
- **Layout mobile-first** — responsivo do mobile ao ultrawide

O projeto foi desenvolvido com a premissa de **zero dependências externas**.

| Camada | Tecnologia |


| Marcação | HTML5 semântico com atributos ARIA |


| Estilo | CSS3 puro — variáveis, grid, flexbox, animações nativas |


| Lógica | JavaScript Vanilla (ES6+) — sem frameworks |


| Persistência | `localStorage` nativo do navegador |


| Fontes | Google Fonts — Inter + Playfair Display |


| Ícones | Emojis nativos (sem dependência de biblioteca) |


| Build | Nenhum — abre direto no navegador |
